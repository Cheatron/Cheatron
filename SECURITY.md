# Security Policy

## ⚠️ Experimental Project Warning

Cheatron is currently in **v0.0.1** and is considered **highly experimental**.
It is not yet production-ready. Use it at your own risk.

## Supported Versions

Since the project is in its early stages, we only provide updates (including security-related ones) for the latest commit on the `main` branch.

| Version       | Supported |
| ------------- | --------- |
| Latest (main) | ✅        |
| < Latest      | ❌        |

## Reporting a Vulnerability

Even though this is an early-stage project, security is important. If you find a critical vulnerability that could harm users, please reach out responsibly:

- **Email:** serkanaksy@proton.me
- **Note:** Please do not open public issues for security exploits until we've had a chance to look at them.

## Disclaimer

As this is an open-source, experimental tool (0.x.x), the developer(s) provide no guarantees regarding stability or security. We appreciate any feedback that helps make the project more robust!
